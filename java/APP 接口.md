## 通用内容
### 响应数据格式
```json
{
    "errorMsg": "请求成功",
    "errorCode": 0,
    "data": {
        "name": "name",
        "id": 1
    }
}
```
- errorCode : 响应码，0 代表请求成功，其他数值代表请求失败。
- errorMsg : 响应失败信息。
- data : 请求的数据。

### 查询与添加
- 查询：默认使用 get 方法。
- 添加：默认使用 post 方法，添加内容时需要验证用户权限，在 header 中添加 id 字段，传入用户 id 。
