# spring 基础

spring 最主要的特点有两个： **IOC** 和 **AOP** 。

## 创建对象的三种方式

- 手动创建 new
- 工厂模式 get
- 注入 set



## IOC
IOC (Inversion of Control) 控制反转：

>本来是由应用程序管理的对象之间的依赖关系，现在交给了容器管理，这就叫控制反转，即交给了IOC容器，Spring的IOC容器主要使用DI方式实现的。不需要主动查找，对象的查找、定位和创建全部由容器管理。

依赖注入的两种方式：

- 构造方法注入
- setter 注入
