- 列表  <span id="1.9"/>  
字符：- + * ，后面跟空格，加内容代表列表；数字`1.`跟空格，加内容代表有序列表，列表有序无序看第一个列表元素，若以1. 开头则下面的为有序列表。
```
 - 二级列表
 1. 二级列表无序
     * 三级列表
         - 四级列表
             1. 有序列表
             2. 有序列表2
             - 有序列表3
             * 有序列表4
             + 有序列表5
```
    - 二级列表
    1. 二级列表无序
        * 三级列表
            - 四级列表
                1. 有序列表
                2. 有序列表2
                - 有序列表3
                * 有序列表4
                + 有序列表5

- 转义字符\<spqn id="1.13">  
Markdown 中有些字符有特殊含义，比如说\*-等，在特殊的位置可以使用`\*`表示。
比如：
```
*表示斜体*
\*这不是斜体\*
```
*表示斜体*  
\*这不是斜体，显示星号\*

    ***另外，MarkDown支持内嵌html,markdown 做不了的可以直接用html实现。***
