## 常用快捷键  
Ctrl + t 查找所有文件  
Ctrl + p 查找所有命令  
Ctrl + \ 打开或关闭目录树  
Ctrl + r 文件内查找变量或者方法  
Ctrl + b 查找打开的文件  
Ctrl + Shift + b 查找上次提交后更改的文件  
## atom 安装插件
- 在 atom 中搜索插件, install；
- 使用 apm install 安装；
- 从 git 下载源码，拖到 user\\.atom\\packages 下面，在当前 package 下面运行命令:
```
apm install
```
第一种和第二种安装太慢，而且没法看进度，第三种方法比较快
